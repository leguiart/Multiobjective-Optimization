

El folder: MOO - Tarea 1 - GA, contiene la implementación del algoritmo genético.
Las pruebas se pueden efectuar corriendo el archivo FunctionOptimizationTests.py de la siguiente manera:
python FunctionOptimizationTests.py
Cada vez que se corren las pruebas se produce una nueva entrada en schaffer_analytics.json y sphere_analytics.json. Dichos archivos contienen
actualmente los datos utilizados en el reporte. 
Si se quiere extraer los resultados del reporte correr el jupyter notebook ResultsAnalysis.ipynb sin correr las pruebas, o bien, si se quiere
obtener nuevos resultados y no afectar los resultados del reporte, mover los archivos .json a otro lado.

El folder: MOO - Tarea 1 - GD, contiene la implementación del gradiente descendiente.
Las pruebas se pueden efectuar corriendo el archivo descenso.py de la siguiente manera:
python descenso.py
Sucede lo mismo para este caso en cuanto a los archivos .json y la obtención de resultados que para el algoritmo genético.
